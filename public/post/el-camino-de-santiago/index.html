El Camino de Santiago

December 27, 2017

Hi. This is the story of my journey across the Camino de Santiago. I’ve done my best to be as accurate as possible to the description of people, events, happenings, dialogues, and places. This is the first time I’ve ever written anything so extensive, and personal. I do not claim to be skilled enough to capture the entirety of my experience, so you’ll have to forgive me if you finish and have any lingering questions – I’m always open to talking about the experience. Enjoy! If you’ve already read the Introduction, start at Beginnings. The story builds upon itself, so it will make the most sense if you start at Introduction.
